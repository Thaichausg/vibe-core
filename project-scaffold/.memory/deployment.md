# Deployment

## Platform
- Local Environment: Windows
- Tools: Batch scripts

## Commands
- Start 9router: `start-9router.bat`
- Start DS2 API: `start-ds2api.bat`
- Start Open Design: `start-open-design.bat`
- Update 9router: `update-9router.bat`

## Release Checklist
- Kiểm tra các port có bị chiếm dụng không.
- Đảm bảo các dependency trong từng sub-folder đã được cài đặt.

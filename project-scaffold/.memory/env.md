# Environment

## Required Variables
> Chỉ lưu tên biến, không lưu giá trị secret.

| Variable | Purpose | Required locally |
|---|---|---|
| `PORT` | Cổng chạy dịch vụ | Yes |
| `API_KEY` | Key truy cập các AI models | Yes |

## Notes
- Các biến môi trường thường được cấu hình trong file `.env` tại từng thư mục con hoặc thông qua hệ thống.

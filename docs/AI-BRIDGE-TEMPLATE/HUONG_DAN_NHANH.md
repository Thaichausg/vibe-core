# Khởi tạo Workflow AI Bridge (AG x CC)

Để áp dụng Workflow "Bộ nhớ chung" này cho BẤT KỲ dự án mới nào, bạn chỉ cần làm theo 2 bước sau:

**Bước 1: Copy Template vào thư mục dự án**
Hãy copy thư mục `.planning` và file `CLAUDE_GUIDE.md` từ thư mục `AI-BRIDGE-TEMPLATE` sang thư mục dự án mới của bạn.

**Bước 2: Bắt đầu Vibe Coding**
- **Quy tắc phân vai linh hoạt**: AG giỏi Design, CC giỏi Logic. Nhưng nếu một người hết quota, người kia sẽ **"Tiếp quản" (Take over)** để làm nốt phần việc của bạn mình.
- Bạn gọi **AG**: *"AG, hãy tạo giao diện trang web. Nếu xong rồi mà tôi gọi CC không được, bạn hãy làm luôn phần logic nhé."*
- Bạn gọi **CC**: *"CC, hãy viết API. Nếu AG hết quota, bạn hãy thiết kế nốt các trang còn lại theo style hiện có."*

Chỉ cần nhắc cả 2 AI **đọc và ghi file .planning/STATE.md**, tụi mình sẽ luôn đồng bộ và có thể thay thế nhau bất cứ lúc nào!
